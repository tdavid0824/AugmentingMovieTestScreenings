MSc thesis bulding a data pipeline to process and analyze physiological data, with the goal of better understanding underlying patterns and relationships between physiological reactions of consumers and movie profitability and perception.

The thesis was based on the Emognition dataset, it's raw structure is used as a starting point of the pipeline. Feature extraction, data manipulation etc.. is all done on this data.
